函数

代码块中存储一段用于处理单任务的代码，然后在任何需要的时候用一个简短的命令来调用
JavaScript有许多内置的函数，浏览器函数不是核心JavaScript语言的一部分而是被定义为浏览器API的一部分

事件

事件是编程时系统内发生的动作或者发生的事情，系统会在事件出现时产生或触发某种信号，并且会提供一个自动加载某种动作，每个可用的事件都会有一个事件处理器,事件处理器有时候被叫做事件监听器
